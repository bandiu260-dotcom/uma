# CareerPilot

## Project Overview

CareerPilot is a Personal Career Management System designed for students.

The application helps students organize their academic and placement journey in one place.

Instead of maintaining notebooks, sticky notes, Excel sheets or remembering everything mentally, CareerPilot allows students to manage everything digitally.

---

## Problem Statement

Students usually face problems like:

- Forgetting assignment deadlines
- Missing coding practice
- Forgetting certification schedules
- Losing placement application details
- Poor goal planning

CareerPilot aims to solve these problems.

---

## Target Users

- Students
- Fresh Graduates

---

## Main Modules

1. Task Management
2. Coding Practice Tracker
3. Certification Tracker
4. Placement Tracker
5. Resume Tracker
6. Goal Tracker
